## TL;DR (by status, 40 rows)

- **FIXOPS_WINS:** 17 — Fixops materially ahead on platform breadth (12-step Brain Pipeline, MPTE 19-phase, FAIL chaos engine, 8 native scanners, 650+ MCP gateway, quantum-safe evidence, 345 engines, 573 routers, 445 UI pages, 36,838 tests, scheduled reports, ServiceNow bidirectional sync, WebSocket event stream, background pipeline workers, RBAC+JWT+SSO, audit-log append-only, dual-mode orchestration/native).
- **PARITY:** 4 — Secrets detection (entropy+regex), pre-commit hook generation, CLI exists for scan/ingest, per-repo enable/disable of rule families.
- **TRUECOURSE_WINS:** 13 — Tiered LLM context router + pre-flight cost modal, unified det+LLM rule taxonomy, violation stable-identity lifecycle chain, zero-infra `.truecourse/` file store, architecture-aware graph (service→layer→module→method with confidence + evidence), diff-mode UI with React Flow dimming, Claude Code skills shipped, committed `hooks.yaml` policy, interactive file-tree + Monaco code viewer, DB detector + ER diagram, traced end-to-end flows with step types, `BaseCLIProvider` Zod-validated `claude --json-schema` stdin pattern, socket.io progress checklist with resume.
- **GAP_EXISTS:** 6 — VS Code extension, YAML/JSON dynamic rule DSL, semantic type-query layer (TS Compiler / Pyright LSP), pre-flight LLM estimate UX, `previousViolationId` identity chain on findings, `.fixops/` zero-infra local store.
- **Proposed new gap rows:** 2 (`NEW-G070` LSP-backed semantic layer, `NEW-G071` IDE-style file tree + Monaco code viewer with violation gutter).

## Methodology (5 lines)

Read `truecourse-analysis.md` end-to-end (sections 1–10, ~300 lines). Cross-referenced `fixops-inventory.md` and `docs/CTEM_PLUS_IDENTITY.md` for Fixops capability claims. For every TrueCourse capability, ran targeted `grep -rn`/`ls` against `suite-core/core/*.py` (345 engines), `suite-api/apps/api/*_router.py` (573 routers), `.omc/prds/v2/*.md` (374 PRDs), and `suite-ui/aldeci-ui-new/src/` — paths and line numbers below are real hits, not inferred. Existing `gap-matrix.md` rows GAP-001..069 (especially the TrueCourse amendment set GAP-061..069) were consulted and cited where applicable. Known limits: did not execute runtime tests; PRD text was not deeply audited for self-consistency; UI grep focused on file-level presence, not render correctness.

## How to read this

- A **node** is one symbol (function, class, module) extracted by graphify. The color reflects the *file* the symbol belongs to, since TrustGraph wiring is a file-level property (one `from core.trustgraph_event_bus import ...` covers all symbols in that file).
- **GREEN** = the file directly imports `core.trustgraph_event_bus` or calls `get_event_bus()`, `bus.emit()`, `bus.publish()`, or `_emit_event()` somewhere.
- **YELLOW** = the file lives under `suite-api/apps/api/` and is therefore auto-covered by `ResponseInterceptorMiddleware`, which sniffs every POST/PUT/PATCH response for entity IDs and emits an event without the router needing to know.
- **RED** = no known wire into TrustGraph. These are the next-priority files to add explicit emits to.
- **GREY** = documentation / config / non-Python — excluded from the % coverage.

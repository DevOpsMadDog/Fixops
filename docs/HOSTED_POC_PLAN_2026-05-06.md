# Hosted POC Deployment Plan — 1-Week Ship

**Objective**: Send prospects a live ALDECI URL within 7 days. Cost-optimized, fastest path.

**Recommendation**: **Fly.io** (single small VM, $10/mo baseline, Docker-native, 15–20 min setup).

---

## 1. Deployment Target Comparison

| Platform | Cost/mo | Setup Time | Notes |
|----------|---------|------------|-------|
| **DigitalOcean droplet** | ~$24 | 30 min | Ubuntu 24.04, manual Docker install, Postgres, nginx. Mature, reliable, but slowest. |
| **Fly.io** | ~$10 base + $0.15/GB RAM | 15–20 min | Docker-native, auto-SSL, custom domains, built-in Postgres. **FASTEST**. Free tier covers first 3 small VMs. |
| **Railway.app** | ~$20 | 10 min | GitHub PR deployments, auto-Postgres, simplest UX. Less control, pricier. |

### Recommendation: **Fly.io**
- **Why**: Fastest time-to-URL, Docker-native (no build translation), cheapest at scale.
- **Path**: `flyctl launch` → secrets → custom domain → 1-day validation.

## 2. Deployment Steps (Fly.io)

### Prerequisites
```bash
# Install flyctl (macOS)
brew install flyctl

# Log in
flyctl auth login

# From repo root
cd /Users/devops.ai/fixops/Fixops
```

### Step 1: Initialize Fly App
```bash
flyctl launch --name aldeci-poc-1
# Prompts: confirm region (SJC/SFO), skip Postgres (we'll use SQLite), skip Redis
```

### Step 2: Set Secrets (from `.env`)
```bash
flyctl secrets set \
  FIXOPS_API_TOKEN="aldeci-poc-demo-$(date +%s)" \
  FIXOPS_JWT_SECRET="$(openssl rand -hex 32)" \
  FIXOPS_MODE="enterprise" \
  OPENROUTER_API_KEY="${OPENROUTER_API_KEY}" \
  ANTHROPIC_API_KEY="${ANTHROPIC_API_KEY}" \
  ALDECI_SEED_DEMO="1"
```

### Step 3: Configure Persistent Volume (SQLite + Memory)
Edit `fly.toml` (auto-generated by `flyctl launch`):
```toml
[mounts]
source = "aldeci_data"
destination = "/app/data"

source = "aldeci_state"
destination = "/app/.fixops_data"

[[volumes]]
name = "aldeci_data"
initial_size = 10

[[volumes]]
name = "aldeci_state"
initial_size = 5
```

### Step 4: Deploy
```bash
flyctl deploy
# Watches build log, shows live URL: https://aldeci-poc-1.fly.dev
```

### Step 5: Custom Domain (Optional, Post-Demo)
```bash
flyctl certs add aldeci.yourdomain.com
# Follow DNS prompt (CNAME to fly.dev)
```

### Step 6: Validation
```bash
curl -s https://aldeci-poc-1.fly.dev/health
# Expected: {"status":"healthy","uptime_seconds":...}
```

## 3. Auth + Multi-Tenant for Prospects

### Option A: Static API Key per Prospect (Simplest)
1. Each prospect gets a unique `FIXOPS_API_TOKEN` at signup (rotate after 30-day trial).
2. UI auto-populates tenant org from JWT `org_id` claim.
3. All queries filtered by `org_id` in FastAPI `Depends(_verify_org_id)`.

**Implementation**: Already wired. No code changes needed. Just vary the token per POC instance.

### Option B: GitHub OAuth (Recommended for 2+ prospects)
1. Register app at https://github.com/settings/apps
2. Set `GITHUB_OAUTH_CLIENT_ID` + `GITHUB_OAUTH_CLIENT_SECRET` secrets.
3. Identity governance engine (`suite-core/identity_governance.py`) already handles org-scoped scope.
4. Users sign in via GitHub → auto-create tenant org based on GitHub org.

**Timeline**: 2 additional hours (identity + OAuth UI flow).

**Recommendation**: Start with **Option A** (1-day faster), upgrade to Option B after first 2 prospects.

## 4. Cost Projection per POC (1 Month Trial)

### Compute
- **Fly.io small VM** (0.25 CPU, 512MB RAM): $10/mo
- **Bandwidth** (typical 10–50 scans/month): included in free tier
- **Volume storage** (250MB SQLite + .swarm/memory.db): $5/mo

**Subtotal**: **$15/mo compute**

### LLM / AI Council
Assuming 1 prospect, ~50 automated scans + 20 dashboard queries/month:
- **OpenRouter (Qwen 3.6 Max)**: ~$15 (tokens: 50 × 5K = 250K tokens)
- **Anthropic Claude** (escalation fallback): ~$15 (tokens: 20 × 2K = 40K tokens)

**Subtotal**: **~$30/mo LLM**

### Total Per Active POC
**~$45–50/mo per prospect** (conservative, includes spikes).

### Monthly Spend (5 Concurrent POCs)
- 5 × $50 = **$250/mo ALDECI**
- Secrets management / observability (minimal): **~$20/mo**
- **Total**: **~$270/mo for 5 concurrent prospects**

## 5. Risks & Blockers

| Risk | Mitigation | Timeline |
|------|-----------|----------|
| **No OPENROUTER_API_KEY** | Founder must provide. Falls back to free Qwen 3.6 (limited). | Day 1 |
| **SQLite single-process** | Fine for 1–2 concurrent users per POC. >5 users → queuing. | N/A (acceptable for POC) |
| **No HTTPS cert automation** | Fly.io auto-provides `*.fly.dev` cert. Custom domain needs 1 hour setup. | Post-demo OK |
| **Memory limit (512MB)** | Sufficient for single prospect. Upgrade to 1GB ($20/mo) if scanning >1K repos. | Day 3 (if needed) |
| **No backup** | Fly.io retains volume snapshots. For production, add daily Postgres + S3. | Post-POC |

## 6. Timeline & Handoff

### Day 1: Deploy First Prospect POC
- **Founder**: Provides OPENROUTER_API_KEY + Anthropic key.
- **CTO**: Run steps 1–6 above. **1 hour elapsed.**
- **Validation**: Load dashboard, run 3-repo scan, check Brain Pipeline logs.
- **URL sent**: Share `https://aldeci-poc-1.fly.dev` with prospect.

### Day 2–3: Testing & Auth
- Prospect imports repos, kicks off scans.
- Monitor Fly.io logs for errors: `flyctl logs`.
- If Option B (OAuth) needed, wire GitHub OAuth (~2 hours).

### Day 4–7: Iterate for 2–3 More Prospects
- Spin up `aldeci-poc-2`, `aldeci-poc-3` (each 15 min).
- Monitor cost/performance.
- If avg. response time >2s, scale POC-1 VM to 0.5 CPU ($20/mo) + 1GB RAM.

---

## 7. Post-POC → Production

Once prospects convert:
1. **Migrate to single enterprise instance** (managed Postgres + Redis, $100–200/mo).
2. **Add multi-org isolation** (already architected, just enforce in routers).
3. **Backups + monitoring** (Fly.io PostgreSQL, Datadog agent).
4. **Custom SLA**: Uptime guarantee, log retention, support tier.

## 5-Minute Walkthrough

### Minute 1: The Problem + Analytics Dashboard [V3] (60s)
**Say**: "Every enterprise runs 5-15 scanners producing 11,300 findings/week. 68% are false positives. ALdeci is the brain that sits above all scanners and makes decisions."

```bash
curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/analytics/dashboard/overview | python3 -m json.tool
```
**Point to**: total findings, deduplication ratio, noise reduction.

### Minute 2: 8 Native Scanners — Air-Gapped [V3+V9] (60s)
**Say**: "ALdeci has 8 built-in scanners. In air-gapped environments — no Snyk, no Semgrep — we scan code directly."

```bash
for scanner in sast dast secrets container cspm; do
  printf "%10s: " "$scanner"
  curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/$scanner/status | python3 -c "import sys,json;print(json.load(sys.stdin).get('status','operational'))" 2>/dev/null
done

# Run live SAST scan
curl -s -X POST -H "X-API-Key: $TOKEN" -H "Content-Type: application/json" \
  -d '{"code":"import os; os.system(input())", "language":"python"}' \
  http://localhost:8000/api/v1/sast/scan | python3 -m json.tool
```

### Minute 3: Brain Pipeline + MPTE Verification [V3+V5] (60s)
**Say**: "Our 12-step Brain Pipeline deduplicates, correlates, and risk-scores. Then MPTE PROVES exploitability with micro-pentests."

```bash
curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/brain/stats | python3 -m json.tool
curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/mpte/stats | python3 -m json.tool
```

### Minute 4: AutoFix + Signed Evidence [V3+V10] (60s)
**Say**: "When we find a real vulnerability, AI generates the fix. Then we create cryptographically signed evidence for auditors."

```bash
curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/autofix/health | python3 -m json.tool
curl -s -X POST -H "X-API-Key: $TOKEN" -H "Content-Type: application/json" \
  -d '{"framework":"SOC2","app_id":"demo-enterprise-app"}' \
  http://localhost:8000/api/v1/evidence/export | python3 -m json.tool
```

### Minute 5: MCP Gateway + Knowledge Graph [V7+V3] (60s)
**Say**: "We're the first AppSec platform AI agents can USE. 705 tools auto-discovered via MCP. Knowledge Graph maps blast radius."

```bash
curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/mcp/tools | python3 -c "
import sys,json;d=json.load(sys.stdin)
t=d.get('tools',d) if isinstance(d,dict) else d
print(f'MCP Tools: {len(t)}')"

curl -s -H "X-API-Key: $TOKEN" http://localhost:8000/api/v1/knowledge-graph/status | python3 -m json.tool
```

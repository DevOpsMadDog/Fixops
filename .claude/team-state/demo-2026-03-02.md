```bash
curl -X POST -H "X-API-Key: $FIXOPS_API_TOKEN" -H "Content-Type: application/json" \
  http://localhost:8000/api/v1/sast/scan \
  -d '{"code":"import sqlite3\ndef login(user,pw):\n  conn=sqlite3.connect(\"app.db\")\n  conn.execute(f\"SELECT * FROM users WHERE name='"'"'{user}'"'"' AND pass='"'"'{pw}'"'"'\")\n","language":"python"}'
```
**Script**: "We just scanned code with our built-in SAST engine. No Snyk, no SonarQube — ALdeci has 8 native scanners: SAST, DAST, Secrets, Container, CSPM/IaC, API Fuzzer, Malware, and LLM Monitor. All work air-gapped."
